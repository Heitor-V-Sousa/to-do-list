# to-do list

Uma aplicação de lista de tarefas desenvolvida com HTML, CSS e JavaScript.

O projeto permite criar tarefas com data e horário, marcar tarefas como concluídas e armazenar todas as informações no navegador utilizando Local Storage, mantendo os dados mesmo após fechar ou atualizar a página.

## Funcionalidades

- Adicionar novas tarefas
- Definir uma data para cada tarefa
- Definir um horário para cada tarefa
- Marcar tarefas como concluídas
- Salvar as tarefas no Local Storage
- Carregar automaticamente as tarefas salvas ao abrir a aplicação
- Interface simples e intuitiva
